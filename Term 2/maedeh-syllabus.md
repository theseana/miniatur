# Data Types (نوع داده)
### String(str, رشته)
- index(پلاک)
- iterate(با حلقه for)
- ', "(اول و اخر رشته)
- '+' (رشته‌ها به هم می‌چسبند)
- میشه برش داد
### Integer(int, عددصحیح)
- '+, -, *, /, ...' (مثل عملگرهای ریاضی)
### Float(float, عدداعشاری)
- '+, -, *, /, ...' (مثل عملگرهای ریاضی)
### Boolean(bool, درست نادرست)
- True, False
- داخل شرط‌ها استفاده میشوند
### List(list, لیست)
- [](اول و اخر رشته)
- index(پلاک)
- iterate(با حلقه for)
- میشه برش داد
### Dictionary(dict, دیکشنری)
- key: value(کلید: ارزش)
- {}(اول و اخر رشته)
- iterate(با حلقه for)
<br>

# Control
### if
- condition(شرظ)
- :(بعد شرط)
- indent(دستورها با فاصله نوشته شود)
- else / elif
### for
- :(اخر)
- indent(دستورها با فاصله نوشته 
- iterate(رشته و لیست range)
### while
- تازمانیکه شرط درست است دستورات را اجرا میکند
- condition(شرظ)
- :(اخر)
- indent(دستورها با فاصله نوشته)
### Function
- def 
- :(اخر)
- indent(دستورها با فاصله نوشته)
- return
- parameter(میتوان به تابع مقدار مختلفی فرستاد)
- call(باید تابع را صدا کرد)
